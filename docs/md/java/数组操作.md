# 数组操作

## 读取

```java
public class ArrayDemo2 {
    public static void main(String[] args) {
        // 静态初始化
        String[] names = new String[]{"a","b","c","d","e","f"};
        System.out.println(names[1]);
    }
}
// b
```

数组，计算机会在内存中为其申请一段 **连续** 的空间，并且会记下索引为 `0` 处的内存地址。

<img src="../../assets/img/image-20220413115854188.png" alt="image-20220413115854188" style="zoom:50%;" />

1. 找到该数组的索引 0 的内存地址： 1111；
2. 将内存地址加上索引值 1，作为目标元素的地址，即 1111+ 1 = 1112，对应的元素为 "b"，这时便找到了目标元素。

3. 时间复杂度是常数级别，为 O(1)

## 查找

通过给定值进行查找，需要遍历数组，进行逐一比对，最多查找 `n` 次，`n` 为数组的长度，时间复杂度为O(N)；

## 插入

1. 通过数组的长度和位置计算出即将插入元素的内存地址，然后将该元素插入到指定位置即可。
2. 将该元素插入到数组中的其他位置，首先需要为该元素所要插入的位置 **腾出** 空间，然后进行插入操作。

## 删除

当数组的长度为 n 时，删除第一个元素

1. 删除操作
2.  移动其余元素 （n - 1）

共需要的步骤数为 1 + (n - 1) = n 步，时间复杂度为 O(N)

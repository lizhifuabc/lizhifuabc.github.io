# з§’жқҖжҙ»еҠЁи®ҫи®Ў

з§’жқҖзҡ„дёҡеҠЎзү№зӮ№пјҡ

  1. жҠҳжүЈд»·ж јгҖӮе•Ҷе“ҒжҲ–жңҚеҠЎйҖҡеёёд»ҘжҠҳжүЈд»·ж јеҮәе”®пјҢиҝҷеҗёеј•дәҶи®ёеӨҡж¶Ҳиҙ№иҖ…гҖӮ
  2. йҷҗе®ҡж—¶й—ҙгҖӮйҖҡеёёеңЁзү№е®ҡж—¶й—ҙж®өеҶ…иҝӣиЎҢпјҢж¶Ҳиҙ№иҖ…еҸӘиғҪеңЁиҝҷж®өж—¶й—ҙеҶ…иҙӯд№°е•Ҷе“ҒжҲ–жңҚеҠЎгҖӮ
  3. йҷҗе®ҡж•°йҮҸгҖӮе•Ҷе“ҒжҲ–жңҚеҠЎзҡ„ж•°йҮҸйҖҡеёёжҳҜжңүйҷҗзҡ„пјҢж¶Ҳиҙ№иҖ…йңҖиҰҒеңЁйҷҗе®ҡж—¶й—ҙеҶ…жҠўиҙӯпјҢд»Ҙе…Қй”ҷиҝҮжңәдјҡгҖӮ
  4. еҝ«йҖҹиҙӯд№°гҖӮйҖҡеёёе…·жңүеҝ«йҖҹиҙӯд№°еҠҹиғҪпјҢж¶Ҳиҙ№иҖ…еҸҜд»Ҙеҝ«йҖҹжөҸи§Ҳе•Ҷе“ҒжҲ–жңҚеҠЎпјҢе№¶еңЁеҮ з§’й’ҹеҶ…е®ҢжҲҗиҙӯд№°гҖӮ
  5. й«ҳе№¶еҸ‘и®ҝй—®гҖӮйҖҡеёёдјҡеҗёеј•еӨ§йҮҸж¶Ҳиҙ№иҖ…пјҢеӣ жӯӨз§’жқҖзі»з»ҹйңҖиҰҒиғҪеӨҹеә”еҜ№й«ҳе№¶еҸ‘и®ҝй—®пјҢдҝқиҜҒж•°жҚ®е®үе…Ёе’Ңзі»з»ҹеҸҜйқ жҖ§гҖӮ

з§’жқҖзҡ„зі»з»ҹзү№зӮ№пјҡ

  1. зһ¬ж—¶е№¶еҸ‘йҮҸйқһеёёй«ҳпјҡзһ¬й—ҙе№¶еҸ‘еі°еҖјйқһеёёй«ҳгҖӮ
  2. иҜ»еӨҡеҶҷе°‘пјҡеә“еӯҳзҡ„жҹҘиҜўи®ҝй—®ж•°йҮҸиҝңиҝңеӨ§дәҺе•Ҷе“Ғзҡ„иҙӯд№°ж•°йҮҸгҖӮ

# дёҡеҠЎжҖқи·Ҝ

йҰ–е…ҲжңүдёӘж ёеҝғзҡ„е®—ж—Ёпјҡз§’жқҖзҡ„жң¬иҙЁдёҚжҳҜеҚ–е•Ҷе“ҒпјҢиҖҢжҳҜдёҖдёӘдәәж°”пјҢиҜҙзҷҪдәҶе°ұжҳҜжІЎеҝ…иҰҒйӮЈд№Ҳи®Өзңҹ

  1. йў„зәҰз§’жқҖпјҢеҸӘжңүйў„зәҰзҡ„дәәжүҚиғҪеҸӮдёҺз§’жқҖ
  2. жҢҮе®ҡз§’жқҖж•°йҮҸпјҢе…¶е®һе°ұжҳҜеҒҮз§’жқҖпјҢдҫӢеҰӮ 100% зҡ„жөҒйҮҸпјҢйҡҸжңәиҝ”еӣһдёҖйғЁеҲҶзӣҙжҺҘеӨұиҙҘпјҲи®°еҫ—еҗҺз«Ҝиҝ”еӣһпјүпјҢж— жүҖи°“еҚ–дёҚеҚ–зҡ„е®ҢпјҢеҸӘиҰҒдёҚи¶…еҚ–е°ұеҸҜд»ҘдәҶгҖӮ

# еёёи§ҒжҠҖжңҜж–№жЎҲ

дёәдәҶеә”еҜ№иҝҷз§Қй«ҳжөҒйҮҸе’Ңй«ҳе№¶еҸ‘зҡ„жғ…еҶөпјҢз§’жқҖзі»з»ҹйҖҡеёёйңҖиҰҒйҮҮз”Ёзү№ж®Ҡзҡ„жҠҖжңҜе’Ңж–№жЎҲжқҘдҝқиҜҒзі»з»ҹзҡ„зЁіе®ҡжҖ§е’ҢжҖ§иғҪгҖӮ

  • зј“еӯҳжҠҖжңҜпјҢдҫӢеҰӮ RedisпјҢжқҘеӯҳеӮЁе•Ҷе“Ғзҡ„еә“еӯҳдҝЎжҒҜпјҢе№¶йҖҡиҝҮиҜ»еҸ–зј“еӯҳжқҘжҸҗй«ҳзі»з»ҹзҡ„и®ҝй—®йҖҹеәҰгҖӮ
  • ж¶ҲжҒҜйҳҹеҲ—жҠҖжңҜпјҢдҫӢеҰӮ RabbitMQпјҢжқҘеӨ„зҗҶеӨ§йҮҸз”ЁжҲ·зҡ„иҙӯд№°иҜ·жұӮпјҢе№¶йҖҡиҝҮејӮжӯҘеӨ„зҗҶжқҘеҮҸиҪ»зі»з»ҹзҡ„еҺӢеҠӣгҖӮ
  • еҲҶеёғејҸжҠҖжңҜпјҢдҫӢеҰӮеҲҶеёғејҸж•°жҚ®еә“е’ҢеҲҶеёғејҸжңҚеҠЎжһ¶жһ„пјҢжқҘжҸҗй«ҳзі»з»ҹзҡ„еҸҜжү©еұ•жҖ§е’Ңе®№й”ҷиғҪеҠӣгҖӮ
  • е®үе…ЁжҠҖжңҜпјҢдҫӢеҰӮеҸҢйҮҚйӘҢиҜҒе’ҢжҺҲжқғи®ӨиҜҒпјҢжқҘйҳІжӯўжҒ¶ж„Ҹз”ЁжҲ·зҡ„ж”»еҮ»е’Ңж¬әиҜҲиЎҢдёәгҖӮ

# жөҒзЁӢеҲҶжһҗ

е…¶е®һеңЁдёүдёӘжөҒзЁӢдёӯпјҢеҸӘжңүз§’жқҖжҳҜе№¶еҸ‘йҮҸжҳҜжңҖеӨ§зҡ„гҖӮеңәжҷҜзҡ„еӨ„зҗҶж–№ејҸжҳҜжҠҠдёӢеҚ•еҒҡжҲҗејӮжӯҘзҡ„гҖӮ

иҝҷз§ҚжөҒзЁӢзңӢзқҖжҳҜжІЎжңүд»Җд№Ҳй—®йўҳпјҢдҪҶжҳҜе®һйҷ…зҡ„жғ…еҶөжҳҜдёӢеҚ•жҳҜжІЎжңүд»Җд№Ҳе№¶еҸ‘зҡ„пјҢиҝҷж ·жҠҠж•ҙдёӘжөҒзЁӢз»ҷжӢүй•ҝдәҶпјҢеҸҜд»ҘиҝӣиЎҢиҝӣдёҖжӯҘзҡ„дјҳеҢ–еҰӮдёӢпјҡ

# зј“еӯҳйў„зғӯ

з§’жқҖзҡ„еңәжҷҜжҳҜ иҜ»>еҶҷпјҢйҒҝе…ҚдәҶеӣ дёәзј“еӯҳдёӯжІЎжңүж•°жҚ®иҖҢйҷҚдҪҺзі»з»ҹжҖ§иғҪзҡ„й—®йўҳгҖӮ

  1. е®ҡж—¶д»»еҠЎе°Ҷ зҠ¶жҖҒдёәе·ІеҸ‘еёғдё”жңӘејҖе§Ӣзҡ„з§’жқҖжҙ»еҠЁгҖҒз§’жқҖе•Ҷе“ҒеҶҷе…Ҙredisзј“еӯҳпјӣ
  2. жү«жҸҸе·ІиҝҮжңҹзҡ„з§’жқҖжҙ»еҠЁпјҢз§»йҷӨзј“еӯҳгҖӮ

# йҷҗжөҒж“ҚдҪң

  1. еҗҢдёҖipйҷҗжөҒ
  2. еҗҢдёҖз”ЁжҲ·йҷҗжөҒ
  3. жҺҘеҸЈиҜ·жұӮж¬Ўж•°йҷҗжөҒ
  4. йӘҢиҜҒз ҒйҷҗжөҒпјҢе»әи®®дҪҝз”Ёж»‘еқ—йӘҢиҜҒз ҒпјҢе®үе…ЁжҖ§иҫғй«ҳгҖӮ

# жүЈеҮҸеә“еӯҳ

  1. ж•°жҚ®еә“пјҡдҝқйҡңдёҚи¶…еҚ–зҡ„жңҖз»ҲжүӢж®ө

еҹәдәҺж•°жҚ®еә“зҡ„д№җи§Ӯй”ҒпјҢиҝҷж ·дјҡе°‘дёҖж¬Ўж•°жҚ®еә“жҹҘиҜўпјҢиҖҢдё”иғҪеӨҹеӨ©з„¶зҡ„дҝқиҜҒж•°жҚ®ж“ҚдҪңзҡ„еҺҹеӯҗжҖ§гҖӮ

<update id="updateSkillRemaining">
        update
            t_skill_goods
        set
            version = version + 1,
            skill_remaining = skill_remaining - 1
        where
            id = #{id}
            and skill_remaining > 0
            and version = #{version}
</update>
  1. Redis lua и„ҡжң¬жүЈеҮҸеә“еӯҳ
local key = KEYS[1]
-- е…ҲеҲӨж–ӯе•Ҷе“ҒidжҳҜеҗҰеӯҳеңЁпјҢеҰӮжһңдёҚеӯҳеңЁеҲҷзӣҙжҺҘиҝ”еӣһ
if (redis.call('exists', key) == 1) then
    local stock = tonumber(redis.call('get', key));
    -- иҺ·еҸ–иҜҘе•Ҷе“Ғidзҡ„еә“еӯҳпјҢеҲӨж–ӯеә“еӯҳеҰӮжһңжҳҜ-1пјҢеҲҷзӣҙжҺҘиҝ”еӣһпјҢиЎЁзӨәдёҚйҷҗеҲ¶еә“еӯҳ
    if (stock == -1) then
        -- ж— еә“еӯҳйҷҗеҲ¶
        return 1;
    end
    -- еҰӮжһңеә“еӯҳеӨ§дәҺ0пјҢеҲҷжүЈеҮҸеә“еӯҳгҖӮ
    if (stock > 0) then
        redis.call('incrby', key, -1);
        -- иҝ”еӣһеү©дҪҷеә“еӯҳж•°
        return stock - 1;
    end
    -- еҰӮжһңеә“еӯҳзӯүдәҺ0пјҢжҳҜзӣҙжҺҘиҝ”еӣһпјҢиЎЁзӨәеә“еӯҳдёҚи¶ігҖӮ
    return 0;
end

return -1;
